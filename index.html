Welcome to Purvi Pinki Games!

This is a fun collection of games for kids of all ages. Here's how to get started:

  • Click on any game tile from the main menu to start playing
  • Use the "Instructions" button in each game to learn how to play
  • Click the sound button in the top right to turn sounds on or off
  • Click "Back to Menu" anytime to return to the main menu

Have fun playing!
